Detailed Description

Holds the prerendered position of a single upmix output channel.

Produced by PrerenderUpmixIndicatorInBounds() for each channel in the selected AudioChannelSet and delivered via onRingPositionsChanged. Contains the pixel position (for painting / overlay consumers) as well as the real-world position used for live-mode DS100 comparison.
