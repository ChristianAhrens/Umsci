Detailed Description

Floating side-panel for storing and recalling a single upmix transform snapshot.

Shares the same tucked/visible side-panel geometry as UmsciDbprProjectComponent. When tucked, only the rightmost grab strip is visible at the window's left edge. The panel sits directly above the dbpr project panel.

Parent responsibilities

The parent must:

  1. Size this component to exactly (s_panelWidth × s_panelHeight).
  2. Set the initial x-position: -(s_panelWidth - s_grabStripWidth) for Tucked, s_panelMargin for Visible.
  3. Subscribe to onStateChangeRequested and animate the x-position when it fires.
  4. Wire onStoreRequested / onRecallRequested to capture and apply the snapshot.
  5. Call setHighlightColour() whenever the app's accent colour changes.
