Detailed Description

Loads and holds a d&b audiotechnik .dbpr project file's data.

Owned by MainComponent. Call loadProjectFromFile() to read a .dbpr SQLite database; on success the internal ProjectData is updated and onProjectLoaded is fired on the JUCE message thread so UI components can update safely.

An empty (default-constructed) ProjectData is held until the first successful load; hasProjectLoaded() distinguishes "nothing loaded yet" from "load failed".
