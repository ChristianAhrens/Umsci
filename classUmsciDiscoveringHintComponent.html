Detailed Description

Fullscreen overlay shown when no DS100 connection has been configured yet.

Displayed by MainComponent in place of UmsciControlComponent while the application is in DeviceController::State::Disconnected and no target IP has been set. Contains a simple label prompting the user to open the connection settings dialog.
